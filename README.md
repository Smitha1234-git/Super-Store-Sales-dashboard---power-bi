# Super-Store-Sales-dashboard---power-bi
Power Bi Super Store Sales Dashboard Project
## Overview
This dashboard is to contribute to the success of a business by utilizing data analysis technique, specifically focusing on time series analysis, to provide valuable insights and accurate sales forecasting.

## Tools used
- Power BI
- DAX

## Key insights
- Dashboard creation to identify KPI's, leverage historic data and apply time series analysis to generate sales forecasts for next 15 days.
- End goal is to share valuable insights and actionable information that can drive strategic decision making- and support the supermarket's goal for growth, efficiency, and customer satisifaction.
